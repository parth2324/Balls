Run the Run.java after compiling it (CMD compilation -> include Ball.java), and Enter as the program asks :
--minimum length for connection.
--no. of balls.
UI :
+ -> speed increase.
- -> speed decrease.
(ctrl) + (+) -> minimum length for ball connection increase.
(ctrl) + (-) -> minimum length for ball connection decrease.
r -> increase (RGB) -> red component.
e -> decrease (RGB) -> red component.
g -> increase (RGB) -> green component.
f -> decrease (RGB) -> green component.
b -> increase (RGB) -> blue component.
v -> decrease (RGB) -> blue component.
Enjoy !
